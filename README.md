**Zenqor CRM streamlines the operational complexities of high-concurrency environments:**

1. **Secure Authentication:** Role-based access control (RBAC) separating Admin privileges from standard Staff via Firebase Auth.
2. **Staff Management:** Centralized directory for all team members across various departments (Administrators, Moderators, Support, Helpers).
3. **Project Tracking:** Monitor development pipelines from initiation to deployment with priority tagging.
4. **Task Delegation:** Assign, track, and archive daily operational tasks efficiently.
5. **Issue & Bug Tracker:** Comprehensive reporting system with severity levels, specific categorization, and real-time status updates.
6. **Responsive UI:** A custom-built enterprise dark-mode design that works flawlessly on desktop, tablet, and mobile devices.

Untuk melancarkan sistem ini, pastikan anda menetapkan konfigurasi Firebase di dalam fail luaran yang selamat. Pastikan kredensial anda tidak terdedah di repositori awam.

*   **Firebase Config:** Cipta fail `firebase-config.js` di direktori utama yang mengandungi API Key, Auth Domain, dan Project ID anda.
*   **Security Check:** Pastikan fail `.gitignore` anda mengandungi `firebase-config.js` untuk melindungi data sensitif pangkalan data.
*   **Launch:** Buka `index.html` menggunakan pelayan web (web server) pilihan anda untuk mengakses portal.
