# Security Policy for Zenqor Technologies

At Zenqor Technologies, we take the security of our infrastructure, scripts, and cloud applications seriously. We appreciate the efforts of the community and security researchers in helping us maintain a highly secure and resilient ecosystem.

## Supported Versions

Security updates and active patches are currently provided for the following major versions of our core systems:

| Version | Supported          | Notes |
| ------- | ------------------ | ----- |
| 2.x.x   | :white_check_mark: | Active development & security patches. |
| 1.5.x   | :white_check_mark: | Critical security patches only. |
| < 1.5   | :x:                | Deprecated. No longer supported. |

*Note: We highly recommend all clients and partners upgrade to the latest 2.x.x release to ensure maximum stability and security.*

## Reporting a Vulnerability

If you discover a security vulnerability within any of Zenqor Technologies' repositories, web applications, or server deployments, please do **not** report it through public GitHub issues or community channels.

Instead, please report it directly to our core team via email:
👉 **Email:** [security@zenqor.com](mailto:security@zenqor.com)

### What to include in your report:
To help us resolve the issue as quickly as possible, please provide:
* A detailed description of the vulnerability and its potential impact.
* Exact steps to reproduce the issue (including any specific configurations or scripts involved).
* Any proof-of-concept (PoC) code or screenshots, if applicable.

### What to expect from us:
1. **Acknowledgement:** We will acknowledge receipt of your vulnerability report within **48 hours**.
2. **Investigation:** Our engineering team will review and verify the issue. We will keep you informed of our progress.
3. **Resolution:** Once the vulnerability is verified and patched, we will notify you immediately before rolling out the update to our production servers and public repositories.

Thank you for helping us keep Zenqor Technologies secure for everyone.
